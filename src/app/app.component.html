MB Attached Notification

Here is a demonstration of how to use the Angular CDK Connected Overlay.

Features

  • The overlay will automatically reposition as you scroll and resize the screen.
  • Using an Intersection Observer, the overlay / notification is hidden when the anchor / 'blue dot' goes out of view.
  • [autoShowNotificationAfterInit] parameter to control if the overlay / notification is automatically displayed upon loading.
  • Content Projection is used to define what the content of the overlay / notification is.
